# Support

- Ask usage and design questions in [GitHub Discussions](https://github.com/CruxExperts/envman/discussions).
- File reproducible defects in [GitHub Issues](https://github.com/CruxExperts/envman/issues).
- Report vulnerabilities privately as described in [SECURITY.md](SECURITY.md).

When reporting a problem, include Envman version, Linux distribution, Python and uv versions, the command you ran, and sanitized output. Never include environment values, backup passwords, API keys, or private machine topology.
